# ATM Simulation Tool

## Overview

The ATM Simulation Tool is a Python-based console application that simulates basic banking operations through a secure ATM interface.

Users can log in using a 4-digit PIN and perform common banking activities such as checking account balance, withdrawing money, and depositing funds.

## Features

- Secure 4-digit PIN Authentication
- Balance Inquiry
- Cash Withdrawal
- Cash Deposit
- Account Data Storage using Text Files
- Menu-Driven User Interface

## Technologies Used

- Python
- File Handling
- Console-Based Interface

## Project Structure

- atm.py – Main ATM interface and user interaction
- bank.py – Banking operations and business logic
- data.txt – Stores account information

## Sample Operations

- Login using a valid PIN
- Check current account balance
- Withdraw money
- Deposit money
- Exit the application securely
